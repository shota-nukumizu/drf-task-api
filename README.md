# Task API for Django REST Framework

Django REST Frameworkで開発したタスク管理アプリ。認証はJWT認証を使用。

**ある企業の社長が、社員に必要なタスクを分担する際に用いる社内のTodoアプリのAPI**を想定して開発。

# インストール

```powershell
django-admin startproject backend .
django-admin startapp task # タスク管理を行う上で必要なタスクをこのアプリで開発する
django-admin startapp partner # 仕事を振る社員のデータをここで保存する
```

# 余談

今回は初の試みとしてDjangoの管理サイトを自分で試しに制作してみた。**正直Djangoの管理サイトの仕組みがよく理解できていないので、管理サイトについてより深く勉強しTwitterやブログ、GitHubでアウトプットしておく必要がある。**

**DRFでViewを設計する際には、予めライブラリ`rest_framework.routers`をインポートしてそこから`DefaultRouters`のインスタンスを作成してそこにURIと`ViewSet`を引数として代入すること**を忘れていた。(ここでエラーが発生して解決に20分くらいかかってしまった...)

Django REST FrameworkはNestJS同様に今後の開発や実務で必ず使うので、基本の内容を再度公式ドキュメントやGitHubで再度確認しておきたい。
